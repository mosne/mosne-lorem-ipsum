Lorem Ipsum Insert Sandbox

Use this page to test the Lorem Ipsum Quick-Insert extension. Click inside any of the fields below to focus, then click the extension icon and select a size.

Instructions

  1. Open Chrome Extensions page: chrome://extensions
  2. Enable Developer mode in the top-right.
  3. Click Load unpacked and select the mosne-lorem-ipsum folder.
  4. Focus on any of the input areas above.
  5. Open the extension and click any button. The generated Lorem Ipsum should be placed exactly where your cursor was.
